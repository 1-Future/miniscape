Combat Debug Plugin — Capturing Real OSRS Data

5 March 2026

How do you build an authentic OSRS experience without access to the source code? You build a spy. The Combat Debug Plugin is a RuneLite plugin that intercepts and logs every meaningful game event in real-time.

Why RuneLite?

RuneLite is the most popular open-source OSRS client. It exposes a rich plugin API that hooks into the game's event system — NPC spawns, combat events, inventory changes, XP gains, animations, and more. It's the botting community's best friend, and now it's ours too.

What the Plugin Does

The Combat Debug Plugin subscribes to RuneLite events and logs structured data:

@Subscribe
public void onHitsplatApplied(HitsplatApplied event) {
    Actor actor = event.getActor();
    Hitsplat hitsplat = event.getHitsplat();
    debugLog(String.format("Hitsplat: %s took %d damage (type: %s)",
        actor.getName(), hitsplat.getAmount(), hitsplat.getHitsplatType()));
}

@Subscribe
public void onAnimationChanged(AnimationChanged event) {
    Actor actor = event.getActor();
    if (actor instanceof Player) {
        debugLog("Player animation: " + actor.getAnimation());
    }
}

@Subscribe
public void onStatChanged(StatChanged event) {
    debugLog(String.format("XP: %s +%d (total: %d, level: %d)",
        event.getSkill(), xpGained, event.getXp(), event.getLevel()));
}

File Logging

Early on, we lost data when RuneLite crashed. Now everything is written to ~/.runelite/combat-debug.log with append mode — sessions accumulate, nothing is lost.

private PrintWriter logWriter;

@Override
protected void startUp() {
    File logFile = new File(
        RuneLiteAPI.RUNELITE_DIR, "combat-debug.log");
    logWriter = new PrintWriter(
        new FileWriter(logFile, true)); // append mode
    logWriter.println("=== SESSION START: " +
        LocalDateTime.now() + " ===");
}

Data Captured So Far

  • 1,474 lines from a complete Tutorial Island playthrough
  • 10 NPC definitions with IDs, positions, and combat levels
  • All skill animations — fishing (621), woodcutting (879), firemaking (733), etc.
  • Combat formulas — melee (386), ranged (426), magic (711)
  • XP values — exact gains for every tutorial action
  • Complete inventory tracking — every item received and used
The plugin doesn't play the game for us. It watches us play and writes down everything that happens. We do the playing — it does the remembering.
