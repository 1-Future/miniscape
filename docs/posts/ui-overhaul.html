Interface

UI Overhaul — Total Panel Freedom

8 March 2026

OSRS players are particular about their interfaces. RuneLite has 200+ plugins largely because people want to control every pixel of their screen. MiniScape now delivers that same spirit — every panel is cloneable, colorable, renameable, and snappable to a configurable grid.

Panel Cloning

Every UI panel now has a + button. Click it, and you get an exact duplicate — fully functional, live-synced with game state. Want 4 minimaps at different zoom levels? Done. 3 chat windows? Sure. 5 inventories around the edges of your screen? Why not.

Cloned panels are first-class citizens:

  • Minimap clones render their own canvas with real-time terrain, player dots, and NPC markers
  • Chat clones receive all messages in real time
  • Sidebar clones have independent tab switching — view inventory and skills simultaneously
  • Combat clones show live HP bars
  • Dialogue clones mirror text and selections

Each clone gets its own taskbar button and can be removed with ×. There's no limit — clone until your GPU taps out.

Panel Customization

In UI unlock mode, every panel now exposes:

  • Text color picker — change the text color of any panel
  • Background color picker — full RGBA background control
  • Label toggle (T) — shows the panel's title bar as a persistent label, visible even outside unlock mode
  • Rename button (✎) — give any panel a custom name
  • Label bar color picker — customize the label bar color independently

All customizations persist in localStorage — your layout survives page refreshes and browser restarts.

Anchor Grid

Instead of freeform chaos, panels now snap to a configurable anchor grid. Cyan lines divide the screen into equal zones with intersection dots marking snap points.

  • Adjustable X columns and Y rows (range 1–20)
  • 14px magnetic threshold — panels snap when dragged near grid lines
  • Grid density controls (+ / -) in the top banner
  • Grid only visible in unlock mode — clean gameplay view
  • Snap works on both position and resize

Square Minimap

The minimap previously used a circular clipping mask (like OSRS). We removed it — the minimap now fills its full rectangular panel. When you can clone and resize minimaps to any dimension, a circle clip just wastes pixels.

Action Progress Bars

Skilling actions now show an RS3-style progress overlay:

  • Contextual icon — 🪓 for woodcutting, ⛏️ for mining, 🎣 for fishing
  • Action name and time remaining
  • Smooth progress bar animated with requestAnimationFrame
  • Progress synced to the server's 4-tick (2.4s) gathering cycle

Dialogue Box

The dialogue panel got a complete rework:

  • Always draggable and resizable — no need to enter unlock mode
  • No auto-timeout — dialogue stays open until dismissed (critical for NPC dialogue selections)
  • Font size controls in the title bar — a to shrink, A to grow
  • Text and background color pickers inline

Grid Toggles

The debug grid system now has separate controls:

KeyToggleWhat It Shows
GTile gridWhite lines on every tile boundary
Shift+GChunk gridBlue lines every 8 tiles, red lines every 64 tiles (region boundaries)
EElevationHeight values overlaid on each tile

What's Next

The interface system is now flexible enough to support whatever comes next — skill-specific panels, loot trackers, XP counters, boss timers. The clone-and-customize architecture means every new panel type automatically gets all these features for free.
